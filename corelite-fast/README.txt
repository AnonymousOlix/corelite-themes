CoreLite Fast
=============

Theme purpose:
A lightweight classic theme for sites that want strong performance defaults without visual complexity.

Highlights:
- Uses a system font stack only.
- Loads one stylesheet and a tiny inline critical CSS snippet.
- No JavaScript is bundled with the theme.

Review notes:
- Functions are prefixed with `clt_`.
- Strings are prepared for translation.
