CoreLite Blocks
===============

Theme purpose:
A lean block theme for Full Site Editing projects that need clean global styles and a few practical starter patterns.

Included FSE files:
- theme.json
- templates/index.html
- templates/single.html
- templates/page.html
- parts/header.html
- parts/footer.html
- patterns/intro-hero.php
- patterns/post-grid.php
