CoreLite Blog
=============

Theme purpose:
A minimal blog theme for writers, journals, and editorial sites that want a quiet, single-column layout.

Included templates:
- index.php
- single.php
- page.php
- header.php
- footer.php

Review notes:
- Uses prefixed `clt_` functions.
- Loads a single stylesheet from `style.css`.
- Uses semantic HTML5, skip links, and translation-ready strings.
- No external assets, trackers, or bundled promotions.
