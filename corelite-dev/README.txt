CoreLite Dev
============

Theme purpose:
A classic starter theme for developers who want clear template hooks and low-opinion defaults.

Highlights:
- Hook-rich templates using `do_action()` around major regions.
- Minimal CSS intended to stay out of the way.
- Commented setup code for easier onboarding.

Review notes:
- No external assets are bundled or loaded.
- Strings are translation-ready.
- Screenshot file is a placeholder and should be replaced before distribution.
